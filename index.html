Установка с помощью графического интерфейса

шаг 1

Перейти на страницку naps2.com

шаг 2

В верхнем правом углу станички найти вкладку Download
Нажмите на неё.

шаг 3

На открывшейся страничке найдите раздел Linux
Найдите строку x64, пакет обязательно должен быть RPM

шаг 4

Нажмите на надпись RPM.
Автоматически начнётся скачивание файла. Файл, по-умолчанию, скачивается в папку Загрузки.

шаг 5

Откройте файл, двойным нажатием на левую кнопку мыши.
Откроется менеджер пакетов dnfdragora и будет предложено выбрать пользователя-администратора с помощью которого будет произведена установка.

шаг 6

Установленное приложение можно найти:
1. Нажать на кнопку Пуск
2. Выбрать раздел Офис
3. Выбрать приложение Naps2

шаг 7

Подключите сканер к приложению.
Нажмите на кнопку Профили или кнопку Новый профиль
В настройках найдите подключенный сканер, укажите параметры, задайте имя, сохраните.

шаг 8

Пример того, как выглядит подключенный сканер.

Установка с помощью консоли.

шаг 1

Вызовите консоль с помощью комбинации клавиш клавиатуры

ctrl + alt + t

шаг 2

В консоли перейдите под учётную запись администратора

su admin //admin - это имя учётной записи администратора. У Вас она может отличаться от примера.

шаг 3

Для поиска приложения наберите команду.

sudo dnf search naps2

шаг 4

Результатом поиска будет выведено название программы naps2.x86_64
Необходимо произвести установку этого приложения.

sudo dnf install naps2.x86_64 //sudo dnf -y install naps2.x86_64, с ключём -y программа не будет задавать вопросов и просто установит приложение

шаг 5

Установленное приложение можно найти:
1. Нажать на кнопку Пуск
2. Выбрать раздел Офис
3. Выбрать приложение Naps2

Удаление приложения с помощью консоли.

шаг 1

Вызовите консоль с помощью комбинации клавиш клавиатуры

ctrl + alt + t

шаг 2

В консоли перейдите под учётную запись администратора

su admin //admin - это имя учётной записи администратора. У Вас она может отличаться от примера.

шаг 3

Наберите команду и выполните её нажав кнопку Enter

sudo dnf remove naps //После слова naps нажмите кнопку TAB. Терминал автоматически допишет название приложения (как на картинке)

Пример успешно удалённого приложения.
